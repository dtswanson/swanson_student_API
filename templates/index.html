Student API Documentation

Endpoints

  • GET /students
    Retrieve all students.
  • GET /students/<id>
    Retrieve a student by ID. Replace <id> with the student's ID.
  • GET /students/random
    Retrieve a random student.
  • POST /students
    Create a new student. Send a JSON object with the student's name.
  • PUT /students/<id>
    Update a student by ID. Replace <id> with the student's ID and send a JSON object with the updated name.
  • DELETE /students/<id>
    Delete a student by ID. Replace <id> with the student's ID.
